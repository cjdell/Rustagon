Rustagon Firmware Tool

Click the "Connect" button to begin. Please specify "Erase device" when prompted.

If you're having trouble installed the firmware, force the badge into DFU mode using the "boop/reboop" buttons labelled on the back:

  1. Hold in "boop" button
  2. Press and release "reboop" button
  3. Release "boop" button

After firmware update is complete. You may need to press the "reboop" button.

Upon first boot, the badge will attempt to format the filesystem. This takes about 3 minutes.

Please see the Git repo for usage instructions.
